## Cách chạy
1. Cài đặt môi trường Python.
2. Cài dependencies:
```bash
pip install -r requirements.txt
```
3. Chạy dashboard:
```bash
streamlit run streamlit_app.py
```

## Nội dung
- `streamlit_app.py`: ứng dụng dashboard tương tác.
- `dataset/`: chứa dữ liệu nguồn.
- `README.md`: hướng dẫn chạy.
